## 遇到问题?

### 问题 1: 后端启动失败

**解决方案:**
```bash
cd backend
python verify.py  # 检查依赖
pip install -r requirements.txt  # 重新安装依赖
```

### 问题 2: 前端启动失败

**解决方案:**
```bash
cd frontend
rm -rf node_modules  # 删除依赖
npm install  # 重新安装
npm run dev  # 重新启动
```

### 问题 3: 无法获取视频信息

**检查:**
- 后端是否正在运行?
- 浏览器控制台是否有错误?
- 网络连接是否正常?

### 问题 4: 下载失败

**可能原因:**
- 视频链接无效
- 网络连接问题
- 视频平台限制

**解决方案:**
- 尝试其他视频链接
- 检查后端终端的错误信息
